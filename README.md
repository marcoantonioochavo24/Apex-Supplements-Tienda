# APEX SUPPLEMENTS – Tienda Online

Proyecto de tienda online de suplementación deportiva y nutrición fitness.

Aplicación web basada en:
- Cliente HTML, CSS y JavaScript (uso intensivo de LocalStorage)
- API REST en PHP para login, validación de carrito y productos vistos
- Datos simulados en ficheros JSON (usuarios y tienda)

## Estructura inicial

- `login.html` – Página de acceso
- `dashboard.html` – Panel principal con productos destacados
- `categories.html` – Listado de categorías y productos
- `product.html` – Ficha de producto
- `cart.html` – Carrito de compra

El objetivo es que el diseño sea moderno, responsive y muy visual, manteniendo el código lo más limpio y organizado posible.
